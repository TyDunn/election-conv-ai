# Setup

- Clone Repo
`git clone https://github.com/TyDunn/election-conv-ai.git`

- Install
`pip install -r requirements.txt`
`python -m spacy download en`
`Install Flask and SQLite`

- Makefile
`make train-nlu`
`make tain-core`
`make cmdline`
`make action-server`
